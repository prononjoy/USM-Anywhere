To Add Assest
Environment--->Assest
Actions--->Quick
Enter Name or IP or FQDN
Scan the newly added asset for details.
Save


Assests can be added through csv file as well.

CSV file must have this format

test1;this is a test assests;;true;true;1.1.1.1
